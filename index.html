Latest Project

Tic-Tac-Toe Game

The Tic-Tac-Toe game built using HTML, CSS, and JavaScript provides a classic and engaging gaming experience. Using HTML for structure, CSS for styling, and JavaScript for functionality, this game allows players to enjoy the timeless challenge of aligning three X's or O's in a row to win. The HTML provides the grid layout, while CSS adds visual appeal with customizable designs and animations. JavaScript handles the logic behind player moves, checks for win conditions, and updates the game state dynamically.

Rock Paper Scissors Game

Designed with simplicity and fun in mind, our Rock, Paper, Scissors game brings the classic playground favorite to your screen. With a sleek HTML layout and CSS styling, the game is visually appealing and easy to navigate. Using JavaScript for interactivity, players can choose their move by clicking on corresponding icons, engaging in an exciting battle of wits against the computer. The game dynamically updates the scores, displaying wins, losses, and draws in real-time.
